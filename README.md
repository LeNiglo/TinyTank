# Project TinyTank

## Techno

* Graphical Client wrote in Java with the Slick2D library.
* Server wrote in Java with Kryogen library.
* Master Server wrote in NodeJS using ExpressJS and MongoDB.
* WebSite build with MeteorJS, totally REST.
